1. Создать репозиторий TXT - на сайте github.com нажимаем repositories далее new
2. Клонировать репозиторий на компьютер - git clone
3. Создать файл new.txt - touch new.txt
4. Добавить под гит - git add new.txt
5. Закоммитить файл - git commit -m "add new file"
6. Отправить файл на внешний репозиторий - git push
7. Отредактировать содержимое файла - vim new.txt
8. Отправить на внешинй репозиторий - git add new.txt; git push
9. Создать файл preferences.txt - touch preferences.txt
10. В файл preferences.txt добавить информацию - vim preferences.txt
11. Создать файл sklls.txt - touch sklls.txt
12. Сделать коммит в одну строку - git commit add . && git commit -m "Commit description
13. Отправить 2 файла на внешинй репозиторий - git push
14. На веб интерфейсе создать файл bug_report.txt - add file; create new file
15. Сохранить изменения на веб интерфейсе - commit changes
16. На веб интерфейсе модифицировать файл bug_report.txt - edit this file
17. Сохранить изменения на веб интерфейсе - commit changes
18. Синхронизировать внешний и локальный репозиторий - git pull
